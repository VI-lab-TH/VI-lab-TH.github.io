Visual Intelligence Laboratory

Welcome to the Visual Intelligence Laboratory (VI-Lab) Webpage. My name is Thanapong Intharah, a lecturer in the Department of Statistics at Khon Kaen University (KKU). I am the Director of this VI-Lab. We work on various Data Science and Machine Learning projects, some of which we spun off to commercial products.

I did my PhD at UCL under supervision of Professor Gabriel Brostow. At UCL, I had been work on using Computer Vision to improve Programming by Demonstration, Learning to Automate GUI Tasks from Demonstration (project page). Before joining KKU, I have founded a Deep Tech startup, TinyEpicBrains, where we work on invisible QR code. I am an Azure Cloud power user.

VI-Lab Research Highlight

Vision domain such as Medical Imaging and Video Analysis are our expertise, but we also worked on Chatbot, NLP, and other domain. The following are list of selected research projects.

Ultrasonography Image Analysis of Human Biliary Tract
(2020-22) Thanapong Intharah Yupaporn Wanna, Prem Junsawang, Anchalee Techasen, Arunnit Boonrod, Attapol Titapun, Vallop Loapaiboon, Nittaya Chamadol, Narong Khuntikeo

We developed an AI to assist doctors and radiologists in diagnosing abnormalities in Human Biliary Tract.

project page / publication / video

Detecting Human with Concealed Gun by Learning from YouTube's CCTV Videos
(2020) Yupaporn Wanna, Pimonpun Chachomphon, Haemwaan Sivarakst, Thanapong Intharah [2nd place: NSC 2020] [1st place: Priminister Digital Award 2020]
(2021) Phattarawat Jaitiang, Nattharit Sawetwaranont, Thanapong Intharah

We developed a dataset by scraping from YouTube and manually annotated them. We then train an AI to detect persons who concealed when they entered the scene.

project page / dataset

Parasite Learning about Medically Important Parasite Eggs with Chatbot
(2020) Chanikan Phadungkit, Siriwan Thanachitboon, Anchalee Techasen, Thanapong Intharah [Final round: NSC 2020] [Bronze medal: YRSS Award 2020]
(2021) Jantraphon Somsuay, Mutita Butakhiaw, Yaowaluk Sompong, Anchalee Techasen, Thanapong Intharah

We developed a chatbot to help associated medical science students learn about 7 medically important parasite eggs.

report / video / dataset / code

Cover Dance Video Evaluation and a Cover Dance Dataset
(2020) Sirirat Cha-in, Pakinee Kriamthaisong, Nattachai Watcharapinchai, Sitapa Rujikietgumjorn, Thanapong Intharah

We developed a korean cover dance dataset and a pose evaluation technique to evaluate dance imitation score


report / dataset / code

Development of NLP Dataset and Machine Leaning Model for Identifying Type of Law of Post in Legal Webboard
(2021) Kannika Wiratchawa, Tanutcha Khunthong, Thanapong Intharah

Tagging questions according to their topics is useful for internet forum management. In this work, we use the Bidirectional Encoder Representations from Transformers (BERT) model to categorize posts from Thai legal internet forums.

report / video / dataset / code

3D Model Reconstruction Of Tooth For Practicing Tooth Restorative Skill Using Photogrammetry Technique
(2021) Phornpirom Lojarat, Thachamin siangeung, Supanita Daduan, Piyaphong Panpisut, Thanapong Intharah

We applied Photogrammetry technique to build models of plastic tooth to help dental students pratice Prosthodontics.
